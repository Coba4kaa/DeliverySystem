Для успешного запуска данного Вэб-приложения необходимо:
1. Выполнить команду git clone https://github.com/Coba4kaa/DeliverySystem
2. Бэкенд можно сразу запускать как проект
3. Для запуска фронта надо перейти в директорию реакт проекта cd .\delivery_system_frontend\
4. Запустить команду npm install (если установлен Node js)
5. Запустить сервер npm run start
